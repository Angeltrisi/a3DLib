Provides a way for developers to use 3D models that have been compiled into .xnb, as well as many helper methods for the display of said 3D models.
This mod must be strong-referenced by developers for proper usage, as it is a library mod.

Have any issues or questions? Check out the Tests folder in the GitHub:
https://github.com/Angeltrisi/a3DLib/tree/master/Tests

Still have issues? Contact me through GitHub! You can also find me in the tModLoader Discord as queueangel.
